twinject (netdex)
=========
https://github.com/netdex/twinject

Please read all documentation in the repository.

-- INSTALLATION --
Extract all files to your Touhou game directory.

-- RUNNING --
Configure twinject.toml according to the game.
Run twinject.exe, which will chainload the game.

-- CONTROLS --
You can use the ImGui window to control twinject, or the following keyboard shortcuts: 

G - Toggle bot
H - Display debug graphics
